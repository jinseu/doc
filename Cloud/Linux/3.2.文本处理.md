### wc

打印文件的函数，字节数等

```
wc [OPTION]... [FILE]...
wc [OPTION]... --files0-from=F
```

1. -c, --bytes 字节数
2. -m, --chars 字符数
3. -l, --lines 行数
4. --files0-from=F 从F中读取用NULL分隔的文件名，F为-表示从命令行读取
5. -L, --max-line-length 最长的行的行数
6. -w, --words 单词数

例如，计算某个目录以及子目录下，所有文件的行数
```
find ./running/ -type f -print0 | wc -l --files0-from=-
```
